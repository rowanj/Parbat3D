Appendix A - Description Of Source Files

Back to Contents

The following is a brief description of all source files that are in the repositry (at the time this document was written). For more information about the source files, please see the relevent parts of this document.

AUTHORS.txt A description of the authors that have worked on the project.
BandInfo.cpp Container Class for information about image bands.
BandInfo.h A header file for BandInfo.cpp.
BatchIniFile.cpp INI file class optimized for quick sucessive reads.
BatchIniFile.h Header for above
ChangeLog.txt A log of the changes that have been made to the repositry.
config.h A set of configuration settings that effect the way the source is compiled.
console.cpp A class used to display debug output on a console window.
console.h A header file for console.cpp.
ContrastAdvWindow.cpp An advanced contrast stretch window - not complemently implemented.
ContrastAdvWindow.h A header file for the above class.
ContrastWindow.cpp A class that displays a contrast stretch window.
ContrastWindow.h A header file for the above class.
CoordinateInfo.cpp Wrapper for coordinate transforms using GDAL (not fully implemented)
CoordinateInfo.h Header for above class.
COPYING.txt Details about the license Parbat 3D is distributed under.
DisplayTab.cpp A class which displays the display settings in the tool window.
DisplayTab.h A header file for the above class.
DisplayWindow.cpp A class which creates a window that OpenGL can draw to.
DisplayWindow.h A header file for the above class.
FeatureSpace.cpp A class which displays a feature space window.
FeatureSpace.h A header file for the above class.
FeatureSpaceGL.cpp A class which handles the OpenGL content of a feature space window.
FeatureSpaceGL.h A header file for the above class.
FeatureTab.cpp A class which displays the feature space tab and handles its events.
FeatureTab.h A header file for the above class.
GLContainer.cpp Wrapper for event handling, used by FeatureSpace.
GLContainer.h A header file for the above class.
GLText.cpp class for simple, arbitrary display of text in an OpenGL context.
GLText.h A header file for the above class.
GLView.cpp Wrapper for OpenGL initialization and control.
GLView.h A header file for the above class.
ImageFile.cpp Abstract data access class, simplifies image file opening, information gathering and reading and scaling blocks of data.
ImageFile.h A header file for the above class.
ImageGL.cpp OpenGL drawing and management class for main image window.
ImageGL.h A header file for the above class.
ImageHandler.cpp Wrapper for all image related classes, (should) contain all data dependant on each open image, to allow multiple images to be open at a time.
ImageHandler.h A header file for the above class.
ImageProperties.cpp Wrapper to GDAL methods for gathering information about an image (dimensions, number of band, type of file, etc.)
ImageProperties.h A header file for the above class.
ImageTab.cpp A class which displays the image tab of the tool window and handles updating its content.
ImageTab.h A header file for the above class.
ImageTileSet.cpp A class which handles the set of tiles that are used when displaying an image.
ImageTileSet.h A header file for the above class.
ImageViewport.cpp Manages and abstracts coordinate transforms for syncronicity between overview and image windows.
ImageViewport.h A header file for the above class.
ImageWindow.cpp A class which displays the image window and handles its events.
ImageWindow.h A header file for the above class.
IniFile.cpp A file which handles reading and writing to the settings file.
IniFile.h A header file for the above class.
main.cpp The main source file of the application. Includes startup code, global variables and handles constructing and deconstructing required objects when loading or closing an image.
main.h A header file for the above file.
MainWindow.cpp A class which handles showing, hiding, enabling and disabling all of the main windows.
MainWindow.h A header file for the above class.
OverviewGL.cpp A class which handles the OpenGL content for the overview window.
OverviewGL.h A header file for the above class.
OverviewWindow.cpp A class which shows the overview window and handles its events.
OverviewWindow.h A header file for the above class.
Parbat3D.dev The Dev-c++ project file for the application.
parbat3D.exe.manifest A file which describes which version of Common Controls to use.
Parbat3D.ico The main icon for the application.
Parbat3D_private.h A file which includes all of the resources for the application.
Parbat3D_private.rc A file which defines all of the resources for the application.
PointsHash.h Consistent header definitions for hash table for Feature Space.
point_texture.c Contains the texture used by the feature space for points (not currently enabled by default)
PrefsWindow.cpp A class which displays the preferences window and updates the settings.
PrefsWindow.h A header file for the above class.
ProgressWindow.cpp A class which displays the loading progress bar.
ProgressWindow.h A header file for the above class.
QueryTab.cpp A class which displays the query tab and handles updating its contents.
QueryTab.h A header file for the above class.
resources.h A file which defines IDs of resources.
ROI.cpp A class which is used to store an ROI.
ROI.h A header file for the above class.
ROIFile.cpp A class which handles reading and writing to an ROI file.
ROIFile.h A header file for the above class.
ROISet.cpp A class which is used to store a set of ROIs.
ROISet.h A header file for the above class.
ROIWindow.cpp A class which creates the ROI Window and handles its events.
ROIWindow.h A header file for the above class.
rsrc.rc A file which defines the application's resources - eg. menu bar and icons.
ScrollBox.cpp A class which creates an automatically scrolling box.
ScrollBox.h A header file for the above class.
Settings.cpp A class which handles reading and writing settings to the settings file.
Settings.h A header file for the above class.
StickyWindowManager.cpp A class which handles snapping windows together and moving connected windows.
StickyWindowManager.h A header file for the above class.
StringUtils.cpp A class that contains some string handling utilities.
StringUtils.h A header file for the above class.
TestUtils.cpp A class that contains some testing utilities.
TestUtils.h A header file for the above class.
ToolTab.cpp A general class which creates a tab that can be displayed in the tool window.
ToolTab.h A header file for the above class.
ToolWindow.cpp A class which creates the tool window and handles switching between tabs.
ToolWindow.h A header file for the above class.
ViewportListener.cpp Abstract class for receiving events about viewport changes (used by OverviewGL and ImageGL.)
ViewportListener.h A header file for the above class.
Window.cpp A class which creates windows, handles their messages and interfaces with the Windows API.
Window.h A header file for the above class.
