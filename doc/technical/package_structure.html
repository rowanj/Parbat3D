Package Structure

The Parbat 3D Project file organizes the source and resource files into several conceptual packages for ease of managment. The groupings of files is intended to be by scope of functionality, however it has turned out that the actual grouping of the finalised version depends largely on the semantics and beaurocracies of such projects.

Feature Space PKG

This contains classes to do solely with creating, managing, and drawing the 3-D feature space window(s).

Image File PKG

This package provides the central ImageFile and ImageProperties classes from which many areas of the project retrieve image data, along with various wrapper and helper classes to better abstract the project from the workings of the underlying GDAL libraries.

Image Manipulation PKG

The Image Manipulation PKG provides the top-level abstraction of the image, it's loading, display, and navigation to the View PKG.

The ImageHandler class provides a wrapper for each instance a file is opened or closed, allowing the allocation and cleanup of system resources to be modularized and contained.

OpenGL

This sub-group contains classes that abstract the manipulation of the graphics hardware sub-system available at runtime, such that higher level classes can operate using logical data-related calls like set_zoom, and set_image_x/set_image_y.

Support

This sub-group contains data managment and coordinate tracking code.

Region of Interest PKG

The Region of Interest package provides high-level functionality for tracking the regions defined by the user, their metadata (name, colour), and the potentially hundreds of entities contained therein.

It also provides the ability to load and save ROIs to file.

Settings PKG

The Settings package exposes high-level functionality for semantically loading and storing user prefenences, such as window locations, tile size, and amount of system memory to use for the tile cache.

Utils

The Utils package contains global support classes.

Console

A simplified, centrally en/dis-abled buffered console output static class to be used for debugging.

StringUtils

String manipulation and managment functionality to make dealing with the various formats required for the libraries and APIs required a little easier.

View PKG

Provides top-level display and event managment functionality for the project.
