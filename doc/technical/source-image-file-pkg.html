console

console is used to output debugging information to a Win32 console. It is not an object, but rather a series of calls to the Win32 API.

Console::write accepts C standard output formatting (cstdio).

StringUtils

StringUtils is a series of trivial functions to concatenate C strings for use in Win32 API MessageBox calls. They are mostly historical, due to console, but are sometimes called by older code.

TestUtils

TestUtils contains a single function, attemptRasterCall. It was a testbed method used to verify if GDAL was working correctly. It is no longer used.
