ImageHandler

The ImageHandler class conceptually provides a state wrapper for all classes that relate directly to the currently open image.

This was to provide abstraction from global variables, such that concurrent instances could be created, allowing for multiple files to be open at a time without special provision.

OpenGL Classes

This sub-section contains classes that deal directly with, or support the OpenGL routines that run the display of image, overview, and text data within the program.

GLText

GLText provides a wrapper to system-specific font functions, and OpenGL transforms required to draw text in a sensible manner to arbitrary screen positions.

GLView

GLView is a wrapper to system-dependant functions relating to creation, destruction, and maintainence of an OpenGL context.

ImageGL

This class manages the functions required to draw the image in the main image window, along with Regions of Interest and tile/texture maintainence and tracking.

Also provided is a simplified interface for adjusting the contrast and brightness of the image displayed (without affecting the data used for pixel operations or feature space generation).

OverviewGL

This class manages the display of the graphics in the overview window, including the main window bounds highlight, used to visualise the current viewport in the overall image.

It also provides an interface for adjusting the displayed brightness/contrast (see: ImageGL).

Support Classes

ImageTileSet

ImageTileSet provides decoding and caching of image tiles as requested, as well as information for translation between image and Level of Detail coordinates, and data about the current level of detail.

Tiles may be requested either in image or LOD coordinates, with up to 3 image bands packed in the order specified.

Conceptually, it may be possible to implement look-ahead caching and/or multi-threading in the cache system herein; which may be one of the largest possible performance increases attainable in this implementation.

ImageViewport

This class provides consistency for manipulations of the viewport between ImageGL and OverviewGL.

It also helps abstract viewport, zoom, and image coordinates.

ViewportListner

Abstract class that provides a regular interface for classes that wish to be notified about viewport or band events.

Currently inherited only by ImageGL and OverviewGL.
