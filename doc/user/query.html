Querying Pixels

Retrieving pixel values from an open image

Parbat 3D allows you to easily retrieve information about a specific pixel by simply hovering the cursor over a pixel in the image window. To query a pixel, follow these steps:

  1. An image must first be loaded. See loading an image

  2. If the Image Window is not already visible, click on the 'Window' menu followed by 'Image'.



  3. If the Tools Window is not already visible, click on the 'Window' menu followed by 'Tools'.



  4. Click on the 'Query' tab within the Tools Window.



  5. Move the cursor to the Image window and hover above a pixel.



  6. The Query Tab will now display data relating to that pixel. The displayed data includes the value of each band (0 - 255) and the x,y coordinates of the pixel (0,0 being the top left).
