Tour of Main Features

Viewing Images

Loading Images

Before you can do anything useful with Parbat 3D, you must first load an image. To load an image, follow these steps:

  1. Click on the ‘File’ menu, followed by ‘Open’.
  2. Choose the file type you are looking for. The default option shows all JPEG, … images. If you can’t see your image file, trying changing the file type to ‘All Files *.*’. 
  3. Select the image file you wish to display and click Ok. After the image has loaded, you should be able to see the whole image displayed in the Overview window. An Image window will also be displayed showing you the top-left position of the image at the default zoom level.

Navigating An Image

The image can be navigated in the Image Window in a number of different ways. Before the image can be navigated, the Image Window must first be highlighted. To highlight the Image Window, click on its title bar.

Panning the image: 

  • Move the scrollbars left/right or up/down.
  • Click and hold the left mouse button, move the mouse and then release the left mouse button
Zooming the image:
  • Move your mouse wheel up/down.
  • Press 'Page Up', or 'Page Down'.

Displaying Different Bands

Parbat 3D allows you to open image files that contain multiple different image bands. By default, Parbat 3D displays the image using the first three bands of the image, which normally correspond to the red, green and blue components of visible light. To change which bands are displayed, follow these steps:

  1. If the Tools Window is not already visible, click on the 'Window' menu followed by 'Tools'.
  2. Click on the 'Display' tab within the Tools Window.
  3. Underneath the 'R' (red) display channel column, click a the radio button that lines up with the image band that you want to display as the red channel. The default is 'Band 1'.
  4. Underneath the 'B' (blue) display channel column, click a the radio button that lines up with the image band that you want to display as the red channel. The default is 'Band 2'.
  5. Underneath the 'G' (green) display channel column, click a the radio button that lines up with the image band that you want to display as the red channel. The default is 'Band 3'.
  6. Click 'Update' to save the changes. The Image and Overview windows will be updated to display the chosen bands.

Regions of Interests

Regions of Interests (ROI's) are a set of areas on an image that are of interest. ROIs can be created by the user and given specific names and colours. Each ROI can consist of multiple different areas, or 'entities' on the image.

Parbat 3D stores a list of ROIs in memory and allows this list to be saved to and from an ROI file, seperate to the image.

Displaying the ROI Window

Before you can use ROIs, the ROI Window must be visible. To make the ROI Window visible, click on the 'Window' menu followed by 'ROI Window'.

Creating a Region Of Interest

To create a Region Of Interest, click on the '+' button within the ROI Window toolbar. An ROI will be added to the ROI Window with a default name and colour.

Creating Entities Within An ROI

There are three different types of entities (areas of interest) within the image that you can create. These are points, rectangles and polygons.

To create a rectange entity, follow these steps:
  1. Select/highlight the ROI that will contain the entity (area of interest).
  2. Click on the rectangle button within the ROI Window toolbar.
  3. Click the left mouse button on the Image Window at the top-left point of the area that you are interesed in.
  4. Click the left mouse button on the Image Window at the bottom-right point of area that you are interested in.
To create a polygon entity, follow these steps:
  1. Select/highlight the ROI that will contain the entity (area of interest).
  2. Click on the polygon button within the ROI Window toolbar.
  3. Click the left mouse button on the Image Window to create the first point of the polygon that you are interesed in.
  4. Keep clicking the left mouse button on the Image Window to create more points as required.
  5. Double-click the left mouse button on the Image Window to end the polygon. The last point will be automatically joined to the first point.
To create a point entity, follow these steps:
  1. Select/highlight the ROI that will contain the entity (area of interest).
  2. Click on the point button within the ROI Window toolbar.
  3. Click the left mouse button on the Image Window at the point within the image that interests you.

Modifying a Regions Of Interest

To change a ROI's colour, follow these steps:
  1. Within the ROI Window, click on the colour box next that appears next to the name of the ROI that you want to change.
  2. Select a colour from the colour dialog box and click 'Ok'.
To change the name of an ROI, follow these steps:
  1. Select the ROI you wish to change by clicking once on its name.
  2. Wait for a couple of seconds.
  3. Click on the name of the ROI again.
  4. Type in a new name.
  5. Press 'enter' to update the name, or 'Esc' to cancel the change.
To delete an ROI, follow these steps:
  1. Select the ROI you wish to delete by clicking once on its name.
  2. Click on the '-' button within the ROI Window toolbar.

Saving Regions Of Interests

The whole list of ROIs can be saved to a file by following these steps:
  1. Click on the save button within the ROI Window toolbar.
  2. Choose a location, type in a filename and click 'Ok'.

Loading Regions Of Interests

A whole list of ROIs can be loaded from a file by following the steps below. Note that ROIs can be loaded on top of a different image to the one that they were originally created on.
  1. Click on the load button within the ROI Window toolbar.
  2. Choose a file and click 'Ok'.

Feature Space

Generating a Feature Space

Parbat3D can generate a Feature Space, or 3D scatter plot, from data contained within the points of an image. To create a Feature Space based on the currently loaded image, follow these steps:
  1. If the Tools Window is not visible, click on the 'Window' menu followed by 'Tools'.
  2. Click on the 'Feature' tab in the Tools Window.
  3. Select which image bands you want to display as the X, Y and Z variables on the scatter plot.
  4. Change the granuality setting if required. Increasing the granuality will reduce the number of points generated and therfore decrease the loading time.
  5. Click the 'Generate' button.

Navigating a Feature Space

The feature space can be rotated by doing the following:
  • Press and hold the left mouse button within the feature space, move the mouse, then release the left mouse button.
The feature space can be zoomed in/out by doing the following:
  • Press and hold the right mouse button within the feature space, move the mouse forward/back, then release the right mouse button.

Changing The Window Layout

Hiding / Showing Windows

The Overview window remains visible all of the time, unless minimized. Clicking on the 'X' close button at the top-right of the window will close the program.

The extra windows can be shown/hidden by using the 'Window' menu, as mentioned throughout this tour. Each window listed under the 'Window' menu has a tick next to it if it is currently visible, or no tick if it is hidden. To change the visibility state of a window, simply click on one of the menu items. If a 'Window' menu item appears to be grayed out or disabled, it can not be used during the current state of the program and therefore can not be shown.

The extra windows can also be hidden by clicking on the 'X' close button button at the top-right of the window.

Snapping Windows

Parbat3D allows each of windows to be moved around independantly, or snapped together and moved around with the Overview Window.

To move a window, press and hold the left mouse button on its title bar, move the mouse, then release the mouse button.

To snap a window to another one, simply move it towards the second window until they touch each other.

To move multiple windows around the screen at the same time, snap them all to the Overview Window and then move the Overview Window with the mouse.

Resizing The Image Window

The Image Window can be resized by pressing and holding the left mouse button down on a corner of the window, moving the mouse, then releasing the left mouse button.

The Image Window can also be maximized by clicking on the maximize button at the top-right area of the window.

Saving The Window Layout

The positions of the main windows are automatically saved and restored when you close and re-open Parbat 3D. See 'Preferences' section for more inforation.

Preferences

Editing Preferences

The preferences can be modified by opening preferences window. To do this, click on the 'Window' menu followed by 'Preferences'. The preferences window will show the current preference settings and allow you to make changes. Clicking the 'Ok' button will accept the modified settings, but, the new settings won't take effect until you close and re-start Parbat3D. See below for a description of the preferences found in the Preferences Window.

The 'Cache size' setting:

Parbat3D loads images in multiple different sections. Each of the sections are partially stored in memory, known as a cache, as they are loaded in order to reduce the amount of data that needs to be read from disk every time the user pans or zooms. The 'Cache size' setting allows the user to specify how much memory should be used to store this cache. The larger this value is, the quicker you may be able to pan and zoom the image. However, setting this value too high could cause your system to run out of available memory. It is recommended to set this value well below the amount of RAM your system has available.

The 'Texture dimension' setting:

The 'Texture dimension' is related to the 'Cache size' setting. It allows the user to set the size of the different sections that are stored in the cache. By setting this to a larger value, the user may be able to pan futher around the image before more data is required to read from the disk. However, if this value is too high, there won't be enough room in the cache to store multiple different sections and Parbat3D will be forced to read from the disk every time you pan or zoom.

The 'Display open file warning' setting:

The 'Display open file warning' setting allows you to turn off the warning box that is displayed when a user attempts to close the program with an image currently open. If this setting is turned on, the user is asked to confirm that they want to quit. If this setting is turned off, no warning is given to the user and the program will quit straight away.

The Parbat3D Settings File

The Parbat3D settings file is stored in the current user's Window's account. This means that multiple different users can use Parbat3D with their own preferences and window layout. If Parbat3D is run while a particular user is logged on, then closed and run again when a different user is logged in, the preferences that it uses may be different.
