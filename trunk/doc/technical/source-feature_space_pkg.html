FeatureSpace

Once instantiated, this class prepares the data to be displayed for a feature space.

It uses the current ROISet to pull out the points to be displayed in the scatter plot, and a local instance of ImageTileSet to get the values of the appropriate bands from the image.

Of the ROISet, it loads all entities of all *active* ROIs.

For each complete ROI, it accumulates the occurences of each combination of point values into a hash table; before calling FeatureSpaceGL to add those points to those to be displayed.

FeatureSpaceGL

This class controls the actual display drawing via OpenGL of the points in the hash tables passed to it by FeatureSpace.

For each table passed to it, it creates an OpenGL display list of vertex points, the opacity of which is weighted around the average the point densities in that hash table.

There is partial support for displaying points as textured sprites, as well as scaling point size according to distance from the camera.

GLContainer

Isolates event handling code interactions with Window class (see: View PKG) from FeatureSpace

PointsHash.h

Seperates include files, structure definitions (for comparison argument) and typedefs for the hash table format used by FeatureSpace and FeatureSpaceGL.

This was placed in a seperate header to isolate errors while porting sample code and finalizing the definition.
