Image Window Shortcuts

The following shortcut keys and controls operate when the image window is active.

Key / Control Function
Up/down/left/right arrow Pan image in the direction pressed
Shift + up/down/left/right arrow Pan image in the direction pressed, to a greater degree
Page up / page down Zoom the image in / out
Mouse moved with the left button down Pan the image in the direction of the mouse
Mouse scroll wheel moved Zoom the image in the direction of the mouse wheel

Feature Space Window Shortcuts

The following shortcut keys and controls operate when the feature space window is active.

Key / Control Function
Up/down/left/right arrow Pan feature space in the direction pressed
Shift + up/down/left/right arrow Pan feature space in the direction pressed, to a greater degree
Ctrl + up/down/left/right arrow Rotate the feature space in the direction pressed
Page up / page down Zoom the feature space in / out
Mouse moved with the left button down Rotate the image in the direction of the mouse
Mouse moved forward/back with the right button down Zoom the image in the direction of the mouse
Mouse moved with the left and right buttons down Pan the image in the direction of the mouse

Global Shortcuts

The following shortcut keys operate when any of the main windows are active.

Key / Control Function
i Hide/Show the image window
t Hide/Show the tools window
r Hide/Show the ROI window
p Hide/Show the preferences Window
c Hide/show the constrast stretch window
d Create an ROI point
s Create an ROI rectangle
a Create an ROI polygon
F1 Load user mannual
